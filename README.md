# HTML Basic Tags

This project demonstrates the most commonly used HTML tags for beginners.

## 📌 Tags Covered

### Headings

```html
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
```

### Paragraph

```html
<p>This is a paragraph.</p>
```

### Line Break

```html
<br>
```

### Horizontal Line

```html
<hr>
```

### Text Formatting

```html
<b>Bold Text</b>
<strong>Important Text</strong>
<i>Italic Text</i>
<em>Emphasized Text</em>
<u>Underlined Text</u>
```

### Links

```html
<a href="https://example.com">Visit Website</a>
```

### Images

```html
<img src="image.jpg" alt="Sample Image">
```

### Lists

#### Ordered List

```html
<ol>
  <li>Item 1</li>
  <li>Item 2</li>
</ol>
```

#### Unordered List

```html
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>
```

### Tables

```html
<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>John</td>
    <td>20</td>
  </tr>
</table>
```

### Forms

```html
<form>
  <input type="text" placeholder="Enter Name">
  <button type="submit">Submit</button>
</form>
```

## 🚀 How to Run

1. Clone the repository.
2. Open `index.html` in your browser.
3. Explore the examples and learn HTML basics.

## 📚 Learning Outcome

* Understand basic HTML structure.
* Learn common HTML tags.
* Build a foundation for web development.
